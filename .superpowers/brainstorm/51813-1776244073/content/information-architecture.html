L1 驾驶舱首页 /dashboard
6大核心指标卡片 → 三条业务线数据对比图 → AI员工活跃度热力图 → 最近任务动态流
点击指标卡片 → 下钻到对应团队看板 | 点击员工名 → 跳转花名册详情
L1 AI员工花名册 /roster
按团队分Tab（管理/设计/生产）→ 员工缩略卡片网格（头像、名称、职位、状态徽标、核心技能标签）
L2 员工详情页 /roster/:id
多Tab页签：
· Profile — 头像、名称、职位、Soul、Identity、个人简介（可编辑）
· Skills — 技能列表、擅长领域、能力等级（可编辑）
· 数据 — 该员工的个人指标仪表盘（任务量、采纳率、准确率趋势图）
· 成长日志 — 版本迭代时间线（V1.0→V2.0，每次升级说明）
L1 生产看板 /production
上半区：实时任务面板（正在执行的任务卡片 → 员工、任务名、进度条、预估完成时间）
下半区：历史任务列表（可筛选、可搜索 → 每行展开可查看产出内容）
L2 任务详情 /production/:taskId
任务基本信息 → 执行步骤时间线 → 产出内容预览/下载 → 关联AI员工
L1 团队组织架构 /org
树状/节点图展示3条业务线 → 24个AI员工的层级归属关系 → 点击节点跳转员工详情
L1 系统设置 /settings
AI员工管理（增删改）→ 指标基准值配置（人工耗时等）→ 用户与权限管理

核心交互路径

路径1：高层巡检
驾驶舱 → 看到采纳率偏低 → 点击下钻 → 看到AI审计官采纳率61% → 点击进入详情 → 查看成长日志，看到V2.0在迭代中
路径2：找人办事
花名册 → 筛选"生产团队" → 找到AI编剧 → 查看Skills确认能力 → （未来）直接发起任务
路径3：生产监控
生产看板 → 看到AI入库员正在执行3D资源入库 → 进度78% → 预计15分钟完成 → 查看历史产出
路径4：投资人展示
组织架构 → 一图看懂24人AI团队 → 驾驶舱看核心ROI → 花名册展示每个角色的深度 → 生产看板证明"真的在干活"
